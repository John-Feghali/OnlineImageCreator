Help

This page will explain StudioKit and answer some FAQs.

About

StudioKit is an image creation website which allows users to freely combine color, images, and text to create a final image.

Features Overview

You can:

Upload Images

Inputted from your files or taken from our image database, use images creatively!

Add Text

Create free floating text boxes with editable text, font, text color, font size, and shadow!

Export & Download

Save your creation as a PNG or a pdf in the exact size you've made it in!!

Save Projects

Save your project to local storage to continue editing your creation later!

How to Use

  1. Step 1: Open the Editor page and start a new project.
  2. Step 2: Add text elements or upload images using the toolbar on the left.
  3. Step 3: Customize your design using the controls panel to edit colors, fonts, sizes, shadows, and backgrounds.
  4. Step 4: Drag elements freely around the canvas to create your layout.
  5. Step 5: Save your project locally or export it as a PNG or PDF.

FAQ

Q: Where are my saved projects stored?

Saved projects are stored using your browser’s local storage.

Q: Can I upload my own images?

Yes. You can upload images directly from your computer or search images using the Pexels API integration.

Q: Can I download my project?

Yes. StudioKit supports exporting projects as PNG images or PDF files.

Q: Do I need to create an account?

No. StudioKit currently works completely in the browser without authentication.
